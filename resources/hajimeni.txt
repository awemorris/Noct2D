PicoPicoScript
===============

PicoPicoScript は 2D ゲームスクリプティングエンジンです。

中身
----

- windows-x86/ :    Windows 32-bit 実行ファイル (Windows XP 以降)
- windows-x86_64/ : Windows 64-bit 実行ファイル (Windows 7 以降)
- windows-arm64/ :  Windows Arm64  実行ファイル (Windows 11 以降)
- macos/ :          macOS アプリ (macOS 10.13 以降)
- ios/ :            iOS プロジェクト (iOS 13.0 以降)
- android/ :        Android プロジェクト (Android 10 以降)
- unity/ :          家庭用ゲーム機向け Unity Plugin
- wasm/ :           ブラウザ用 WebAssembly
- DOCS/ :           ドキュメント
- SAMPLES/ :        サンプルゲーム
