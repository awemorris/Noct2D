PicoPicoScript
===============

PicoPicoScript は2Dゲームスクリプティングエンジンです。

中身
----

- windows-x86/ :    Windows 32-bit 実行ファイル (Windows XP以降)
- windows-x86_64/ : Windows 64-bit 実行ファイル (Windows 7以降)
- windows-arm64/ :  Windows Arm64  実行ファイル (Windows 11以降)
- macos/ :          macOS アプリ (macOS 10.13以降)
- ios/ :            iOS プロジェクト (iOS 13.0以降)
- android/ :        Android プロジェクト (Android 10以降)
- unity/ :          家庭用ゲーム機向け Unity Plugin
- wasm/ :           ブラウザ用 WebAssembly
- DOCS/ :           ドキュメント
- SAMPLES/ :        サンプルゲーム
