PicoPico Script
===============

"PicoPico Script" is a game scripting engine for 2D games.

Contents
--------

- windows-x86/ :    Windows 32-bit EXE       (Windows XP+ 32-bit)
- windows-x86_64/ : Windows 64-bit EXE       (Windows 7+ 64-bit)
- windows-arm64/ :  Windows Arm 64-bit EXE   (Windows 11 Arm64)
- macos/ :          macOS Binaries           (macOS 10.13+)
- ios/ :            iOS Xcode Template       (iOS 13.0+)
- android/ :        Android Studio Template  (Android 10+)
- unity/ :          Unity Plugin             (Game Consoles)
- wasm/ :           WebAssembly              (Browsers)
- DOCS/ :           Documents
- SAMPLES/ :        Example Games
