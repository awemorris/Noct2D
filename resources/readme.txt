This is the Software Development Kit (SDK) for Noct2D,
a portable 2D game engine powered by NoctLang.

Contents
--------
- windows-x86/ :    Windows 32-bit EXE       (Windows 2000 and later)
- windows-x86_64/ : Windows 64-bit EXE       (Windows 7 and later)
- windows-arm64/ :  Windows Arm 64-bit EXE   (Windows 10 and later)
- linux-x86_64/ :   Linux x86_64 Binaries    (Ubuntu 22.04 and later)
- linux-arm64/ :    Linux Arm64 Binaries     (Raspberry Pi)
- macos/ :          macOS Binaries           (macOS 10.13 and later)
- ios/ :            iOS Xcode Template       (iOS 13.0 and later)
- android/ :        Android Studio Template  (Android 10 and later)
- unity/ :          Unity Plugin             (Game Consoles)
- wasm/ :           WebAssembly              (Browsers)
- manual/ :         Documents
- sample/ :         Example
